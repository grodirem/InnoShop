# InnoShop

1) docker-compose up --build
2.1) ProductService http://localhost:7002/swagger/index.html
2.2) UserService http://localhost:7001/swagger/index.html

4) после подтверждения почты нужно залогиниться
4.1) либо можно использовать готовые учетки (user@1.1 Pass1!; admin@1.1 Pass1!)
