# InnoShop
Инструкция по запуску
1) docker-compose up --build
2) ProductService http://localhost:7002/swagger/index.html
3) UserService http://localhost:7001/swagger/index.html 
4) после регистрации, вместо настоящей почты, сообщение о подтверждении почты выводится в логах user-service
5) после подтверждения почты нужно залогиниться
6) либо можно использовать готовые учетки (user@1.1 Pass1!; admin@1.1 Pass1!)
